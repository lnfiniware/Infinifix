# Security Policy

Repository: `https://github.com/lnfiniware/Infinifix`

## Supported Versions

Current release line: `v0.2 (beta)`.
Security fixes land on `main`.

## Reporting a Vulnerability

Open a private security advisory in:
`https://github.com/lnfiniware/Infinifix/security/advisories`

Please include:

- impact summary
- reproduction steps
- affected distro/kernel
- logs or report bundle if possible

Do not open public issues for sensitive vulnerabilities.
